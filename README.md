# 🚀 Product App (MERN Stack)

A full-stack Product Management App built using MERN stack (MongoDB, Express, React, Node.js).

## ✨ Features
- ➕ Add Product
- ✏️ Edit Product
- ❌ Delete Product
- 📋 View Products List
- 💰 Price Management

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- API: REST API

## 📂 Project Structure
client/        → React frontend  
server.js      → Backend server  
package.json   → Dependencies

## 🌐 API Endpoints

- GET /products → Get all products  
- POST /add → Add product  
- DELETE /delete/:id → Delete product  
- PUT /update/:id → Update product
